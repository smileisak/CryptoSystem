This is the first version of CryptoSystem which is an online platform

for learning cryptography over examples.\n
```
*Language : python2.7*
*Framework: Django1.7*
```

To test the  Beta version just:
```
 	*$ virtualenv --no-site-packages -p /usr/bin/python2.7
	*$ pip install django
	*$ git clone 
```
```
	*$ cd CryptoSystem/
	*$ ./manage.py runserver
```
And Enjoy !
